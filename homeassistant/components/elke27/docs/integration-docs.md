## Installation parameters

When you add the integration, you are asked for these values:

- `Host`: The IP address or hostname of the Elk E27 panel.
- `Access code`: The panel access code used to authorize the connection.
- `Passphrase`: The panel passphrase used to authorize the connection.
- `Panel`: If discovery finds multiple panels, select the one you want to set up.

The integration uses the default panel port `2101` and does not ask you to set it.

## Configuration parameters

There are no configuration options after setup. If you need to change the
connection details, remove the integration and add it again.

## Actions

This integration does not provide custom actions.
