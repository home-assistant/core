## Proposed change

Add reauthentication flow to the Eve Online integration.

When an OAuth2 token expires or is revoked, the integration now triggers a reauth flow so the user can re-authenticate their character without removing and re-adding the config entry.

This includes:
- `async_step_reauth` and `async_step_reauth_confirm` in the config flow
- `ConfigEntryAuthFailed` handling in the coordinator for auth errors
- Reauth-specific strings and translations
- Full test coverage for reauth flow and account mismatch scenarios
